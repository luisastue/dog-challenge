# Dog Challenge
Utilize the https://dog.ceo/dog-api/ to create a fun web app.

##Requirements:
- [x] npx create-react-app dog-challenge
- [x] use redux 
- [x] fetch & display data from the dogs api
- [x] react hooks

##Optional

- [x] filter dogs
- [ ] "Favorite" dogs
- [ ] sort dogs
- [x] slick animations haha I tried...
